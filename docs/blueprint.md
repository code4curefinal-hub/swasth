# **App Name**: Swasthya

## Core Features:

- User Authentication: Secure login and signup functionality for users.
- Vaccination Drive Information: Display information about vaccination drives, including locations and schedules.
- Visiting Camps Details: Provide details about visiting health camps, including timings and available services.
- Medical Notifications: Send medical notifications and alerts to users.
- User Health Records: Allow users to access and manage their health records.
- Multilingual Support: Provide information and access in multiple languages based on user preference.

## Style Guidelines:

- Primary color: Soft green (#7DB954), conveying health and wellness.
- Background color: Desaturated green (#E5F0DD), a light and airy backdrop that reinforces a feeling of health.
- Accent color: Blue-green (#54B98B), provides contrast on calls to action, and emphasizes the app's focus on both physical and emotional wellness.
- Body and headline font: 'PT Sans', a modern and warm sans-serif font.
- Use simple, clear icons related to health and location for easy navigation.
- A clean and intuitive layout with a focus on easy access to key information.
- Subtle transitions and animations to enhance user engagement without being distracting.
